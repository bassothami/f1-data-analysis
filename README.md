# F1 Data Analysis Project

Este projeto utiliza dados reais da Fórmula 1 através da API da Jolpica para calcular métricas de desempenho dos pilotos.

---

## O que o projeto faz

- Coleta dados de corridas de Fórmula 1 via API
- Calcula ranking de pilotos com base em pontos
- Calcula média de posições de chegada
- Calcula consistência dos pilotos (desvio padrão)
- Exporta os resultados para um arquivo CSV

---

## Tecnologias utilizadas

- Python
- Pandas
- NumPy
- Requests (consumo de API)
